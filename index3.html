The Command Line

Sometimes referred to as the command screen or a text interface, the command line is a user interface (any device or program that enables the user to communicate with the computer) that is navigated by typing commands at prompts, instead of using the mouse. (Typing a word or set of words that cause the computer to perform a certain operation)

The Mac command line is a program called Terminal. It lives in the /Applications/Utilities/ folder. Windows comes with a command line, but it is non-standard and more difficult to use. Babun is a free, easy to install command line program.

The command line is a very useful tool when it comes to web development, particularily at times in which you are uploading or managing different computer files and applications.
