Launching a Website

  1. Build Your Website using your prefered Web Editor.
  2. Purchase a Domain: Some places you can register for a free domain include:
    • Freenom
    • DotTK
  3. Find a Web host to host your website.
  4. Upload Files to webhost using File Tranfer Protocol (FTP) tool, such as FileZilla. An FTP is simply a commonly used protocol for exchanging files over the Internet.

Publishing a Website Via Github

GitHub is a "social coding" site. It allows you to upload code repositories for storage in the Git version control system. You can then collaborate on code projects, and the system is open-source by default, meaning that anyone in the world can find your GitHub code, use it, learn from it, and improve on it. GitHub has a very useful feature called GitHub Pages, which allows you to expose website code live on the Web.

  • First, Sign up for GitHub and verify your email address.
  • Next, you need to create a repository for your files to go in.
  • On this page, in the Repository name box, enter username.github.io, where username is your username. Also check Initialize this repository with a README and then click Create repository.
  • Drag and drop the content of your website folder into your repository and then click Commit changes. Always include index.html file
  • Now navigate your browser to username.github.io to see your website online.
