jQUERY

Originally released in January 2006 by John Resig; jQUERY is arguably the most well known Javascript Library. jQuery takes a lot of common tasks that require many lines of JavaScript code to accomplish, and wraps them into methods that you can call with a single line of code. jQuery also simplifies a lot of the complicated things from JavaScript, like Event-handling and DOM transversing and manipulation.

Downloading jQUERY

Just like we need a <link> tag to connect our HTML and CSS, we need a <script> tag to connect our HTML and jQuery. The tag looks like this: 

 <script type="text/javascript" src="script.js"> </script>

jQuery Variables

Like containers; Variables are a place for us to store information for use at a later time. Variables can hold any type of information you work with. A single = sign is used to assign values. For instance, in jQuery, you can write: 

var lucky = 7;
var name = "Codecademy";
var $p = $('p');

jQuery Syntax

jQuery starts up like this: 

$(document).ready(function);

  • $() turns the document into a jQuery object
  • Putting document between the parentheses tells us that we are working on that HTML document.
  • .ready(); is a function, or basic action, in jQuery. It will run the code inside its parentheses as soon as the HTML document is ready.
  • function(){} is the action that .ready() will perform as soon as the HTML document is loaded.
  • .ready(); ends with a semicolon. This tells jQuery that you're done giving it a command.

jQuery Selectors

jQuery selectors allow you to select and manipulate HTML element(s). jQuery selectors are used to "find" (or select) HTML elements based on their name, id, classes, types, attributes, etc. The element name passed to jQuery as identical to the CSS element, only wrapped in quotes. All selectors in jQuery start with the dollar sign and parentheses: $(). 

 $("p") Selects elements based on element name
	$(".class") Selects elements based on Class
	$("#test") Selects elements based on ID
	$("p.intro") Selects all <p> elements with class="intro"

jQuery Events

Events are all the different visitor's actions that a web page can respond to. Some common events include:

To assign a click event to all paragraphs on a page, you can do this: 

 $("p").click();

The next step is to define what should happen when the event fires. You must pass a function to the event: 

 $("p").click(function(){
  // action goes here
});

jQuery Effects

$(document).ready(function() {
    $('thingToTouch').event(function() {
        $('thingToAffect').effect();    
        });
});

The .effect() is the thing that will happen to the selected HTML element. The things that go in the bracket are the conditions of the effect (ex. the time it takes to complete effect.) Effects include: 

$(selector).hide(speed,callback);	Hides selected element.
$(selector).show(speed,callback);	Shows selected element.
   	   $(selector).toggle(speed,callback);	Allows you to toggle between methods.
 $(selector).fadeIn(speed,callback);	Fades in selected element
       $(selector).fadeOut(speed,callback);  	   Fades out selected element
	  	  $(selector).fadeTo(speed,opacity,callback);  	Fades element to specified opacity
	  $(selector).slideDown/Up(speed,callback);	Slides element up or down 
	  	 $(selector).animate({params},speed,callback);	Lets you create custom animations

jQuery DOM Manipulation

DOM stands for Document Object Model. The DOM consists of every element on the page, laid out in a hierarchical way that reflects the way the HTML document is ordered. jQuery allows developers to easily traverse the contents of the DOM in order to find the nodes, elements, or values they're aiming to retrieve.

jQuery also allows us to manipulate the DOM without having to find each element in our html file. For example you can add and remove classes from HTML elements using:

 $(selector).addClass('classname'); Allows you to add a class
$(selector).removeClass('classname'); Allows you to remove a class
