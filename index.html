HTML

First developed by Tim Berners-Lee in 1990; HTML stands for Hypertext Markup Language. Where Hypertext means “text with links in it” and a Markup Language is a programming language used to make text do more than just sit on a page: it can turn text into images, links, tables, lists, and much more.

CSS

CSS stands for Cascading Style Sheets. A style sheet is a file that describes how an HTML file should look. These style sheets are cascading because the sheets can apply formatting when more than one style applies. CSS was first proposed in October 1994 by Håkon Wium Lie, and was published two years later in December 1996.

jQUERY

Originally released in January 2006 by John Resig; jQUERY is arguably the most well known Javascript Library. jQuery takes a lot of common tasks that require many lines of JavaScript code to accomplish, and wraps them into methods that you can call with a single line of code.
